body

string

required

Text of the message.

channel

object

required

Channel to send the message to.

channel object

contactName

string

Name of the contact to send the message to. If a contact already exists for the channel then their name will be updated.

locationUid

uuid

required

Podium unique identifier for location.

senderName

string

The name that will be credited with sending the message. This will appear under the sent message in the Podium app.

setOpenInbox

boolean

Defaults to false

new conversations will be created in a 'closed' state by default. This will override that setting going to 'open' inbox.
